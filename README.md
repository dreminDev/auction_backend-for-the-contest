# Auction Backend

Backend система для проведения аукционов с поддержкой многораундовых ставок и автоматической выдачей подарков.

## Основные возможности

- **Многораундовые аукционы** — гибкая настройка количества раундов и призов
- **Антиснайпинг** — автоматическое продление раунда при ставках в последние секунды
- **Система подарков** — автоматическая выдача с уникальной нумерацией
- **Управление балансами** — транзакционная обработка всех операций
- **Логирование** — полная история всех операций

## Быстрый старт

```bash
# Клонирование
git clone https://github.com/user/auction_backend-for-the-contest.git
cd auction_backend-for-the-contest

# Запуск через Docker (frontend + backend + MongoDB)
docker-compose up -d

# бекенд на http://localhost:5000
```
